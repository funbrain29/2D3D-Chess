# 2D3D-Chess

## Keybinds

```
1 = 3D Perspective View
2 = 2D Orthographic View
3 = 3D Top-Down View

r = Toggle Camera Rotation (Only Visible 3D Perspective View)

Spacebar = Start/Restart Animation
```
